# 🩸 Maze & Mace (미궁과 메이스)

**Maze & Mace**는 React, TypeScript, TailwindCSS로 개발된 고품질 바이오-오가닉(Bio-Organic) 다크 판타지 액션 로그라이크 게임입니다. 플레이어는 미궁의 촉수와 오염물질로 가득 찬 살점 가득한 던전을 탐험하며, 묵직한 메이스를 휘둘러 기괴한 생명체들을 물리치고 오염을 정화하여 미궁 밖으로 탈출해야 합니다.

---

## 🎮 게임 개요 (Game Overview)

미궁은 계속해서 살점처럼 자라납니다. 플레이어는 방을 정화할 때마다 사방에 위치한 유기체 확장 모듈(Expansion Module)을 타격하여 던전을 확장하고 더 깊은 구역으로 전진해야 합니다. 하지만 깊어질수록 미궁의 **오염도(Corruption Level)**가 플레이어의 육체를 잠식하며 시야를 어둡게 만들고 괴물들을 광폭화시킵니다. 욕망을 이겨내고 탈출 포탈을 열어 미궁을 정화하는 것이 최종 목표입니다.

---

## 🛡️ 캐릭터 클래스 빌드 (Character Builds)

플레이어는 각기 다른 장비와 공격 파라미터, 전용 특수 스킬(Q 키)을 가진 4개의 독창적인 캐릭터 클래스를 선택할 수 있습니다.

| 클래스 (Build) | 사용 장비 & 특징 | 특수 스킬 (Skill) | 스태미너 밸런스 |
| :--- | :--- | :--- | :--- |
| **Breaker (파괴자)** | 중형 가시 메이스. 넉백이 강력하며, 적을 벽에 충돌시켰을 때 발생하는 **벽 슬램(Wall Slam) 대미지**가 2배로 증폭됩니다. | **지진 강타 (Earthquake Slam)**<br>지면을 강타하여 넓은 광역 피해를 주고 넉백시킵니다. | 공격/스킬 소모 큼 |
| **Striker (습격자)** | 쌍수 쾌속 검. 대시 시 **완전 무적 프레임(Invincible Frames)**을 가지며 공격 속도가 매우 빠릅니다. | **잔상 대시 (Afterimage Dash)**<br>앞으로 돌진하며 이동 경로의 적들에게 연속적인 잔상 타격을 입힙니다. | 공격 소모 매우 적음 |
| **Cartographer (개척자)** | 유적 탐사 메이스. 방 확장 모듈 타격 시 **스태미너 소모가 60% 감소**하며 미탐사 지역의 격자 타입을 선예측(Forecast)할 수 있습니다. | **유적 파동 (Relic Pulse)**<br>고대 에너지를 방출하여 적들의 속도를 50% 둔화시키는 장막을 형성합니다. | 전반적으로 균형 잡힘 |
| **Guardian (수호자)** | 거대 대검. permanent **35% 대미지 경감 효과**를 상시 보유하여 높은 안정성을 제공합니다. | **회전 베기 (Spinning Slash)**<br>제자리에서 휠윈드를 돌며 적의 투사체를 파괴하고 다단히트 피해를 줍니다. | 방어 및 유지력 우수 |

## ⚙️ 핵심 게임 시스템 (Core Systems)

### 1. 결정론적 데일리 챌린지 모드 (Daily Challenge Mode)
- **무서버 시드 생성**: 매일 다른 오늘 날짜(`YYYY-MM-DD`)를 기반으로 한 32-bit `Mulberry32` 결정론적 PRNG 시드를 사용합니다.
- **동일한 맵 & 스폰**: 모든 플레이어가 당일 플레이 시 정확히 동일한 맵 레이아웃, 방 종류, 몬스터의 생성 좌표를 마주하게 됩니다.
- **데일리 규칙 (Daily Modifier)**:
  - *끝없는 갈증 (Exhaustion)*: 모든 행동의 스태미너 소모량 2배 증가.
  - *심해의 어둠 (Claustrophobia)*: 플레이어 전용 시야 반경 50% 축소.
  - *가속되는 오염 (Corrosion)*: 실시간 오염도 증가 속도 2배 증가.
  - *피의 광란 (Frenzy)*: 모든 몬스터의 이동 속도 30% 증가.
  - *유리 대포 (Glass Cannon)*: 아군/적군 가하는 대미지 모두 2배 증가.
  - *버려진 성소 (Desolation)*: 회복의 성소에서 체력 재생 및 정화 불가능.
- **점수 공유 기능**: 결과 화면에서 챌린지 날짜, 규칙, 콤보, 랭크 등을 포함한 텍스트 템플릿을 클립보드로 즉시 복사하여 공유할 수 있습니다.

### 2. 점진적 스테이지 & 오염 시스템 (Stage & Corruption)
- **3개 스테이지**: Stage 1(미궁 진입부, 오염 한계 25%), Stage 2(중층, 오염 한계 60%), Stage 3(심층 심장부, 오염 한계 100%).
- **오염도별 연출**:
  - *51% ~ 75%*: 몬스터들이 1.35배 커지고 붉은 아우라를 뿜어내는 **엘리트 변종**으로 스폰됩니다.
  - *76% ~ 100%*: 시야가 좁아지고 화면에 끈적한 피막 Vignette와 오염 안개가 드리우며 심층 공포감이 극대화됩니다.
  - *90% 이상*: BGM이 테이프가 늘어난 것처럼 울렁이며 왜곡(Tape Warble)되고 화면이 심장 박동 소리에 맞춰 격렬히 진동합니다.
- **보스 구역**: 각 스테이지의 특정 방 수 확장 시 강력한 전용 보스 몬스터가 출현하며 극적인 연출과 함께 전용 BGM으로 전환됩니다.

### 3. 실시간 통계 HUD & 타깃 에임 어시스트
- `Tab` 키를 꾹 눌러 실시간 생존 시간, 킬 수, 방 개수, 최고 콤보 및 장착하고 있는 장비 모듈 카드를 확인할 수 있습니다.
- 모바일 가상 조이스틱 조작 환경 및 저사양 접근성을 돕기 위해 주변 적을 탐색해 조준을 유도하는 **자동 조준 보조(Auto-Aim Assist)** 시스템을 내장하고 있습니다.

### 4. 사운드 시스템 고도화 (Advanced Web Audio API)
- **공간 음향 (Spatial Audio)**: 실시간 몬스터와 플레이어 간의 거리 및 각도를 계산하여 가까울수록 소리가 커지고 좌우 팬(`StereoPannerNode`) 채널이 찢어지는 현장감을 줍니다.
- **동적 BGM 전환**: 탐색 중(Explore), 일반 전투(Combat), 보스전(Boss) 상태에 따라 드럼 비트 소스의 밀도가 변조되며 자연스럽게 BGM 세션이 전환됩니다.
- **볼륨 세부 저장**: BGM과 SFX 음량을 독립적으로 설정 가능하며 `localStorage`에 자동 저장됩니다.

### 5. 접근성 옵션 (Accessibility Suite)
- **색맹 모드 (Colorblind Mode)**: 붉은색 위주의 오염도 바, 화면 가장자리 Vignette 외곽 경고선 등을 식별하기 편한 청록색(Cyan/Blue) 계열로 치환합니다.
- **화면 흔들림 끄기**: 격렬한 피격/타격 연출 시의 카메라도 진동을 억제하여 멀미를 방지합니다.
- **파티클 감소**: 화면에 비산하는 유기체 파편 및 스파크 개수를 1/2로 클램프하여 최적화합니다.
- **텍스트 크게**: UI 크기 및 데미지 플로팅 텍스트 폰트를 크게 확대합니다.
- **고대비 모드**: 인터페이스 외곽에 두꺼운 흰색 테두리를 강화하고 배경을 검은색으로 강제하여 시인성을 확보합니다.

## 🛠️ 기술 스택 (Tech Stack)

- **프레임워크**: React 19, TypeScript, Vite
- **스타일링**: TailwindCSS, Vanilla CSS (Bio-organic vein keyframes)
- **렌더링**: Canvas 2D API (이펙트, 조명 마스크, 충돌 판정) + SVG Overlay Layer (고품질 캐릭터 애니메이션)
- **사운드 엔진**: Web Audio API (합성음 향상, 로우패스 필터 주파수 스윕, 피치 쉬프터 오실레이터)

## 🚀 시작하기 (Getting Started)

### 설치 (Installation)
프로젝트 디렉토리에서 패키지를 설치합니다.
```bash
npm install
```

### 로컬 개발 서버 실행 (Development Server)
로컬에서 테스트 서버를 기동합니다.
```bash
npm run dev
```

### 배포 빌드 (Production Build)
정적 HTML 번들 파일 하나로 인라이닝 빌드합니다. (Vite SingleFile 플러그인 탑재)
```bash
npm run build
```
빌드에 성공하면 `dist/index.html` 단일 파일이 생성되어 브라우저에서 서버 없이 독립 실행이 가능합니다.
